# Gambling Platform Admin Dashboard

A professional Vue 3 + Tailwind CSS admin dashboard for managing a comprehensive gambling platform. This project includes layout scaffolding, module pages, mock data, routing, and state management with Pinia.

## Highlights
- Responsive sidebar + header layout with dark/light mode toggle.
- Dashboard cards, tables, and charts for real-time operations.
- Mock data for users, transactions, risk alerts, and reports.
- Role-based access control example and multi-language structure.

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
  assets/           Tailwind base styles
  components/       Reusable UI components
  composables/      Shared logic (theme, toasts, realtime)
  data/             Mock data sources
  pages/            Route pages
  router/           Vue Router configuration
  stores/           Pinia stores
```
